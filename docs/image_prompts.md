# Exercise Illustration Prompts

AI image generation prompts for all 60 progression step illustrations.

## Style Guide

All images share the same global style. Prepend this to every prompt below:

```
Flat design athletic illustration, minimalist cartoon figure (gender-neutral),
dark background color #111111, figure in white/light grey, orange accent
color #FF5722 used for motion arrows or highlights, simple bold shapes with
no facial details, clean confident lines, sports training app icon style,
square 1:1 format, no text or labels.
```

## File Naming Convention

Generated images must be saved to:

```
assets/images/exercises/{exerciseTypeName}_{step:02d}.jpg
```

| Exercise     | TypeName    | Example filename        |
|--------------|-------------|-------------------------|
| 伏地挺身     | `pushUp`    | `pushUp_01.jpg`         |
| 深蹲         | `squat`     | `squat_01.jpg`          |
| 引體向上     | `pullUp`    | `pullUp_01.jpg`         |
| 舉腿         | `legRaise`  | `legRaise_01.jpg`       |
| 橋式         | `bridge`    | `bridge_01.jpg`         |
| 倒立撐       | `handstand` | `handstand_01.jpg`      |

After saving all files, add the following line to `pubspec.yaml` under `flutter → assets`:

```yaml
    - assets/images/exercises/
```
